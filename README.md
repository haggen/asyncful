**Asyncful** is a jQuery plugin to make forms AJAX capable.

Licensed under http://creativecommons.org/licenses/by/3.0/

Usage:

    <form ... data-sync></form>

or

    $('form').asyncful({ ... })

Look at $.ajax docs for possible settings: http://api.jquery.com/jQuery.ajax/
